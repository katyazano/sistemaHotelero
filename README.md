# Sistema de Gestión Hotelera

## Descripción del Proyecto
Este proyecto es un Sistema de Gestión Hotelera diseñado para centralizar las operaciones de reservas, inventario de habitaciones y generación de reportes financieros. Permite la interacción eficiente entre Administradores, Personal de Recepción y Huéspedes.
